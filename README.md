## Running the project
To run this project you will need [flutter](https://flutter.dev/) and [git](https://git-scm.com/) installed.

```$bash
# Clone this repository
$ git clone https://github.com/femoraes0/wallet-ui.git

# Access new folder
$ cd wallet-ui

# Install packages
$ flutter packages get

# Run
$ flutter run
```
